# 🏠 Бот «ГЕОРИС»
ИИ-консультант Антон - помощник эксперта Пархоменко Юлии Владимировны по согласованию перепланировок.

## 📋 Описание проекта

Telegram-бот для автоматизации первичных консультаций по перепланировкам квартир и коммерческой недвижимости, построенный на современном стеке технологий:

- **Aiogram 3.x** — асинхронный фреймворк для Telegram Bot API
- **YandexGPT / Router AI** — генерация ответов консультанта; опционально **Yandex AI Agent API** (Агент-Шпион, Агент-Антон) для лидов и экспертных ответов
- **RAG-система** — поиск в базе знаний (80+ документов), интеграция с квизом (предварительное заключение)
- **SQLite + aiosqlite** — персистентное хранение (пользователи, лиды, spy_leads, заявки квиза)

## 🎯 Возможности

1. **Онбординг с согласиями** — запрос согласия на обработку данных и контакта
2. **Квиз (заявка)** — сбор информации (город, тип объекта, этажность, площадь, статус, описание, план); после отправки — предварительное заключение от Агента-Антона и уведомление эксперту
3. **Диалоговый режим** — консультации с учётом истории разговора (RAG, база знаний)
4. **Умный Охотник (GEORIS v2.0)** — карточки горячих лидов (hotness ≥ 3) в рабочую группу с кнопками «Ответить экспертно» (Агент-Антон) и «Взять в работу» (контакт Юлии)
5. **Инвест-калькулятор** — оценка потенциала квартиры под перепланировку
6. **База знаний** — 80+ документов по нормам, законам, процедурам; Yandex AI Agents (Шпион, Антон) при наличии `YANDEX_AI_AGENTS_ENDPOINT`

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка переменных окружения

Скопируйте `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
```

Отредактируйте `.env`:

```env
# Telegram Bot Configuration
TELEGRAM_TOKEN=your_bot_token_from_@BotFather

# Yandex Cloud Configuration
YANDEX_API_KEY=your_yandex_api_key
FOLDER_ID=your_yandex_folder_id

# Database Configuration
DATABASE_PATH=database/bot.db

# Leads Group Configuration
LEADS_GROUP_CHAT_ID=-1003370698977
THREAD_ID_KVARTIRY=0
THREAD_ID_KOMMERCIA=0
THREAD_ID_DOMA=0

# Admin Configuration
ADMIN_ID=your_admin_user_id

# Channel Configuration
CHANNEL_ID=@your_channel_id
```

### 3. Запуск бота

```bash
python main.py
```

## 🔑 Получение токенов и ключей

### Telegram Bot Token

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте полученный токен в переменную `TELEGRAM_TOKEN`

### Yandex Cloud API Key

1. Зарегистрируйтесь в [Yandex Cloud](https://cloud.yandex.ru/)
2. Создайте каталог (folder)
3. Получите API-ключ в разделе "Сервисные аккаунты"
4. Скопируйте `YANDEX_API_KEY` и `FOLDER_ID`

### Admin ID

Ваш Telegram User ID можно узнать у [@userinfobot](https://t.me/userinfobot)

## 🛠️ Разработка

### Добавление нового handler

1. Создайте файл `handlers/new_handler.py`
2. Определите router:

```python
from aiogram import Router
router = Router()

@router.message(...)
async def handler_name(message: Message):
    # Ваш код
    pass
```

3. Зарегистрируйте в `handlers/__init__.py`:

```python
from .new_handler import router as new_router
__all__ = [..., 'new_router']
```

4. Подключите в `main.py`:

```python
from handlers import ..., new_router
dp.include_router(new_router)
```

### Файлы сессий Telethon (`*.session`)

Файлы с расширением `.session` (например, `anton_parser.session`) **не попадают в репозиторий** — они в `.gitignore`. Причины:

- **Безопасность:** в них хранятся данные авторизации Telegram (сессия пользователя). Попадание в репозиторий может скомпрометировать аккаунт.
- **Локальность:** сессия привязана к номеру телефона и устройству; на другом компьютере или после смены пароля файл бесполезен или мешает.

**Что делать при первом запуске парсера чатов (chat_parser, геошпион и т.п.):**

1. В `.env` задайте `API_ID`, `API_HASH` и `PHONE` (номер в формате +79xxxxxxxxx).
2. Запустите создание сессии один раз **на той машине, где будет работать парсер:**

   ```bash
   python session_manager.py
   ```

   Или при первом запуске скрипта, который использует Telethon (например, `chat_parser.py`), сессия создастся автоматически — введите код из Telegram и при запросе пароль 2FA.

3. Файл `anton_parser.session` появится **только локально**. Не добавляйте его в Git и не копируйте на другие машины — там нужно создать свою сессию тем же способом.
