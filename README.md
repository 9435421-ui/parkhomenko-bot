# ТЕРИОН — Экосистема согласования переустройства

## Архитектура
- **Основной бот**: @terion_bot (Aiogram 3)
- **Контент-бот**: @domGrad_bot (Telebot)
- **Единая БД**: SQLite (unified_leads)

## Квиз: 7 шагов
1. Город (свободный ввод)
2. Тип объекта: Квартира / Коммерция / Дом
3. Этаж и этажность (свободный ввод)
4. Площадь, кв.м (свободный ввод)
5. Статус: Планируется / Выполнена
6. Описание изменений (YandexGPT / свободный ввод)
7. Наличие плана: Да / Нет + загрузка файла

## Deep-linking
- `/start` — стандартный вход
- `/start?start=quiz` — сразу квиз, НО с обязательным приветствием и согласием

## Роли команды
- **Джемини**: Техрук, архитектура, инфраструктура
- **Антон**: ИИ-ассистент, клиентская коммуникация

## Установка и запуск
1. Клонируйте репозиторий.
2. Создайте и активируйте виртуальное окружение.
3. Установите зависимости: `pip install -r requirements.txt`.
4. Настройте `.env` файл на основе `.env.example`.
5. Запустите основного бота: `python main.py`.
6. Запустите контент-бота: `python content_bot_mvp/main.py`.
