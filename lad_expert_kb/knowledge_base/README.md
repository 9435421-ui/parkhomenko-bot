# База знаний по перепланировкам

Проект: ИИ‑консультант по перепланировкам и согласованиям  
Назначение: единая база для обучения ИИ, консультантов и продавцов.

## Структура папок (предметные знания)

- `01_общедомовое_имущество/` — общие части дома, УК, ТСЖ и т.п.  
- `02_ТСЖ_и_согласования/` — порядок согласований через ТСЖ, УК, собрания.  
- `03_коммерческая_недвижимость/` — помещения, нежилые объекты, аренда и т.д.  
- `04_шаблоны_документов/` — образцы заявлений, согласий, уведомлений.  
- `05_нормативная_база/` — ФЗ, кодексы, СП, региональные акты.  
- `06_кейсы_судебная_практика/` — разборы дел и выводы для практики.  
- `07_Помещения_и_элементы/` — кухни, санузлы, балконы, лоджии, коридоры и т.д.  
- `08_Процедуры_и_согласование/` — алгоритмы согласования до/после, Москва/МО/РФ.

## Папка ИИ‑консультанта

- `09_ИИ_консультант/01_FAQ_клиенты.md` — типовые вопросы клиентов и эталонные ответы.  
- `09_ИИ_консультант/02_Сценарии_диалогов.md` — пошаговые сценарии (диагностика, продажа, сопровождение).  
- `09_ИИ_консультант/03_Промпт_ИИ_консультанта.md` — системный промпт/профиль ИИ‑консультанта.  
- `09_ИИ_консультант/04_ТЗ_на_RAG_и_бота.md` — техническое задание на интеграцию базы с ботом и RAG.

## Как использовать

1. ИИ‑консультант в первую очередь обращается к файлам из папок 05, 07, 08.  
2. Для общения с клиентом использует структуры из `09_ИИ_консультант`.  
3. Все изменения вносятся через VS Code и синхронизируются с репозиторием/хранилищем.
