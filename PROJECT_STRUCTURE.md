## Структура ТЕРИОН

### Корень проекта
- `main.py` - Точка входа для основного бота (@terion_bot).
- `config.py` - Центральная конфигурация (токены, ID топиков).
- `parkhomenko_bot.db` - Основная база данных (unified_leads).
- `requirements.txt` - Список зависимостей.

### Папки
- `handlers/` - Обработчики сообщений для основного бота.
  - `start.py` - Приветствие, согласие, сбор контакта.
  - `quiz.py` - 7-шаговый квиз.
  - `dialog.py` - ИИ-консультация после квиза.
  - `admin.py` - Админ-панель и CRM.
- `services/` - Бизнес-логика и внешние интеграции.
  - `loyalty_service.py` - Поздравления и праздники.
  - `vk_service.py` - Интеграция с ВКонтакте.
- `utils/` - Вспомогательные утилиты.
  - `notifications.py` - Отправка заявок в топики (2, 5, 8, 88).
  - `yandex_gpt.py` - Интеграция с YandexGPT.
  - `voice_handler.py` - STT (Yandex SpeechKit).
  - `moderation.py` - Анти-мат и фильтрация.
- `database/` - Модули для работы с БД.
  - `db.py` - Асинхронный клиент SQLite.
- `keyboards/` - Определения клавиатур.
- `content_bot_mvp/` - Исходный код контент-бота (@domGrad_bot).
- `docs/` - База знаний и документация.
- `knowledge_base/` - Гео-данные и технические детали объектов.
