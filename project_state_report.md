# Отчет о состоянии проекта (Project State Report)
**Дата**: 02.02.2026

## Текущий статус
- **Бренд**: ТЕРИОН (Terion). ИИ-помощник Антон выступает от имени эксперта.
- **Запуск**: Оба бота запущены в фоновом режиме через `run_bot.sh` и `run_main_bot.sh`.
- **База данных**: Инициализирована `parkhomenko_bot.db` с таблицей `unified_leads`.
- **Конфигурация**: Переменные окружения в `.env` проверены, токены активны.

## Компоненты
1. **Основной бот (@terion_bot)**: Aiogram 3.x. Поддерживает квиз, диалоговый режим (RAG), инвест-калькулятор.
2. **Контент-бот (@domGrad_bot)**: Telebot. Поддерживает создание постов, репортажный режим, планирование публикаций и квиз.
3. **RAG-система**: Подключена к YandexGPT, использует базу знаний в `docs/` (83 документа).

## Операционный статус
- **Lead Flow**: Работает. Лиды сохраняются в БД и дублируются в Telegram-группу.
- **Content Flow**: Работает. Реализован планировщик и поддержка VK-интеграции.
- **Conflict Management**: Ошибка 409 устранена. Исправлены ImportError, связанные с устаревшими модулями (expert, price).
