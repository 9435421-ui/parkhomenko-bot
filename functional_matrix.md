# Функциональная матрица проекта ТОРИОН

Матрица соответствия реализованного функционала требованиям.

| ID | Модуль | Статус | Комментарий |
| :--- | :--- | :--- | :--- |
| F1 | Контент-бот (Workflow) | **OK** | Реализован в `content_bot_mvp`, поддерживает добавление постов. |
| F2 | Планировщик публикаций | **OK** | Используется библиотека `schedule`, есть логика автопостинга. |
| F3 | Генерация текста | **OK** | Интеграция с OpenAI/OpenRouter и YandexGPT. |
| F4 | Генерация изображений | **PARTIAL** | Есть заглушка для DALL-E, отсутствует Yandex Art. |
| F5 | Репортаж (Фото → Пост) | **NO** | Обработка медиафайлов выдает сообщение о будущих обновлениях. |
| F6 | Квиз → Лид → БД | **OK** | Реализован 8-шаговый сбор данных, запись в SQLite и уведомление в группу. |
| F7 | Роли (RBAC) | **PARTIAL** | Ограничено проверкой `ADMIN_ID`, детальные роли (Author/Editor) не внедрены в коде. |
| F8 | Мультиканальность | **PARTIAL** | `VKService` является заглушкой (stub). Основная работа в TG. |
