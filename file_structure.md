# Таблица структуры проекта ТЕРИОН

| Путь к файлу / папке | Описание и роль в системе |
| :--- | :--- |
| `main.py` | Точка входа для основного бота (Aiogram). Инициализация и запуск. |
| `config.py` | Загрузка настроек из `.env` и хранение глобальных констант. |
| `.env` | Файл переменных окружения (токены, API-ключи). *Теперь отслеживается.* |
| `parkhomenko_bot.db` | Единая база данных SQLite. Хранит пользователей и лиды. |
| `run_main_bot.sh` | Скрипт для фонового запуска основного бота с авто-рестартом. |
| `run_bot.sh` | Скрипт для фонового запуска контент-бота с авто-рестартом. |
| `handlers/` | **Папка обработчиков (роутеров):** |
| `  ├── start.py` | Команда /start, согласие ОПД, получение контакта, логика переходов. |
| `  ├── quiz.py` | Реализация 10-шагового квиза для квалификации лидов. |
| `  ├── dialog.py` | Режим ИИ-консультанта (RAG) на базе YandexGPT. |
| `  ├── invest.py` | Инвест-калькулятор для оценки доходности перепланировки. |
| `database/` | **Папка работы с БД:** |
| `  └── db.py` | Асинхронные методы для работы с таблицами `users` и `unified_leads`. |
| `utils/` | **Папка вспомогательных модулей:** |
| `  ├── yandex_gpt.py` | Интеграция с Yandex Cloud (GPT-модели). |
| `  ├── voice_handler.py`| Обработка и транскрибация голоса через SpeechKit. |
| `  └── knowledge_base.py`| Система RAG для поиска по документам компании. |
| `keyboards/` | **Папка интерфейса:** |
| `  └── main_menu.py` | Определения Inline и Reply клавиатур для навигации. |
| `content_bot_mvp/` | **Папка контент-бота:** |
| `  ├── main.py` | Код контент-бота (Telebot) для генерации постов и квиза. |
| `  └── .env` | Настройки специфичные для контент-бота. |
| `docs/` | База знаний: PDF/текстовые документы для обучения ИИ. |
| `uploads/` | Папка для хранения файлов, загруженных пользователями (БТИ и др.). |
| `ARCHITECTURE.md` | Полное техническое описание архитектуры и связей. |
| `project_state_report.md`| Актуальный отчет о работоспособности всех систем. |
