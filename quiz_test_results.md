# Результаты тестовых прогонов квиза (Quiz Test Results)

Проведено виртуальное тестирование (код-ревью потока данных) для двух основных сценариев.

## Тест 1 — Сценарий «Планирую»

*   **Вход:** `/start?start=torion_main`
*   **Путь:** Согласие → Собственник → Москва → Жилое → **Планируется** → Нет сложностей → Для жизни → БТИ есть → Можно подождать → +79990000001
*   **Результат:**
    *   **Лид в БД:** Успешно сохранен в `unified_leads`. `source_bot="qualification"`, `details` содержит `_payload="torion_main"`.
    *   **Уведомление админу:** Сформировано корректно, содержит статус «Планируется».
    *   **Контент:** Выдан стандартный чек-лист документов.
    *   **Завершение:** Бот выдал финальное сообщение с обещанием связи эксперта.

## Тест 2 — Сценарий «Уже сделал»

*   **Вход:** `/start?start=domgrand`
*   **Путь:** Согласие → Собственник → СПб → Жилое → **Уже выполнена** → Стены → Инвест → БТИ частично → Срочно → +79990000002
*   **Результат:**
    *   **Лид в БД:** Успешно сохранен. `_payload="domgrand"`.
    *   **Уведомление админу:** Содержит статус «Уже выполнена» — критически важно для менеджера по продажам.
    *   **Контент:** Выдан **тот же самый** чек-лист документов.
    *   **Завершение:** Аналогично Тесту 1.

## Технические метрики

| Параметр | Статус | Комментарий |
| :--- | :--- | :--- |
| Сохранение payload (источник) | **Успешно** | Позволяет отслеживать эффективность рекламных каналов. |
| Сбор телефона | **Успешно** | Текстовый ввод работает стабильно. |
| Уведомление в группу | **Успешно** | Используется `ADMIN_GROUP_ID`. |
| Анонимность эксперта | **Успешно** | Упоминается «Наш эксперт» без личных имен. |

## Вывод по тестам
Квиз технически надежен: данные не теряются, уведомления приходят, юридическая чистота соблюдена. Однако, **отсутствие вариативности контента** (один и тот же чек-лист) снижает ценность квиза для клиентов, уже выполнивших перепланировку, так как им более актуальна информация об административных штрафах и процедуре легализации (амнистии).
