# Рекомендации по следующим шагам развития ТОРИОН

На основе аудита и поставленных целей (мультибот).

---

## 1. Ближайшие приоритеты (Quick Wins)

1. **Внедрение Репортажного режима (F5):**
   - Реализовать `handle_files` для сохранения фото в `uploads/`.
   - Создавать запись в `content_plan` (status='idea' или 'draft') при получении изображения.
   - Реализовать формулу: `PHOTO + META → IMAGE_ID → POST_ID`.

2. **Запуск Модуля Генерации Изображений (F4):**
   - Интегрировать Yandex Art как первичный источник (PRIMARY).
   - Оставить OpenAI DALL-E как резервный (FALLBACK).
   - Поддержка MODE_1 (topic → image) и MODE_2 (photo → enhanced).

## 2. Развитие архитектуры (Multi-bot)

- **Централизация БД:** Перейти на единую базу `parkhomenko_bot.db` для обоих ботов.
- **Ролевая модель (F7):** Добавить таблицу `admins` с уровнями доступа (1-Author, 2-Editor, 3-Admin).
- **Мультиканальность (F8):** Реализовать активные методы в `VKService` для дублирования постов на стену сообщества.

## 3. Стабилизация

- Перенести мониторинг из `run_bot.sh` в `systemd` сервис для промышленной эксплуатации.
- Исправить импорты в основном `main.py` (модули `expert` и `price`).
