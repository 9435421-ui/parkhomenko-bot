# 🎬 ГЕОРИС — Быстрый старт

## Что это?
Видео-продакшен для контента по перепланировкам. Загрузил видео → Напечатал тему → БОТ сам всё обработал и опубликовал.

## Как использовать?

### Шаг 1: Отправить видео
```
/video
```
Отправить видео (MP4, MOV или любой другой формат)

### Шаг 2: Написать тему
```
Согласование перепланировки в новостройке
```

### Шаг 3: БОТ обработает
БОТ автоматически:
- ✅ Удалит шум и паузы
- ✅ Ускорит видео (1.2x)
- ✅ Добавит ватермарк ГЕОРИС
- ✅ Сгенерирует экспертный текст
- ✅ Опубликует в TG и VK

**Время обработки:** 2-5 минут (зависит от размера видео)

## Что происходит за сценах?
```
ВИДЕО-ОБРАБОТКА (5-10 сек)
├─ Удаление шума (noise reduction)
├─ Удаление пауз (silence removal)  
├─ Ускорение видео (1.2x)
└─ Результат: чистое, динамичное видео

ДОБАВЛЕНИЕ ВАТЕРМАРКА (3-5 сек)
└─ Логотип ГЕОРИС в нижний угол

ГЕНЕРАЦИЯ КОНТЕНТА (3-5 сек)
├─ Анализ темы через Яндекс GPT
└─ Экспертный текст 150-200 слов

ГЕНЕРАЦИЯ ПРЕВЬЮ (2-3 сек)
└─ JPG thumbnail для социальных сетей

ПУБЛИКАЦИЯ (5-10 сек)
├─ Telegram (2 канала)
└─ VK группа ГЕОРИС
```

## Поддерживаемые форматы

✅ MP4 (H.264)  
✅ MOV (Apple)  
✅ WebM  
✅ MKV  
✅ AVI  

**Максимальный размер:** 2 ГБ

## Параметры обработки видео
