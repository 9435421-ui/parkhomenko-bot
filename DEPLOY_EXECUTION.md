## Внесённые изменения

| Пункт | Выполнено |
|-------|-----------|
| **Квиз** | `t.me/georis_bot?start=quiz` запускает квиз (Бриф). В `handlers/start.py`: сообщение «Начинаем квиз (Бриф)», согласие ПД → цепочка вопросов. |
| **Фото → Профи-пост** | Промпт: «Ты — ведущий эксперт GEORIS». Анализ фото (планировка, демонтаж, ремонт), технические нюансы (МЖИ, несущие стены), пост с призывом к действию. |

| **Интересный факт** | При наличии записей в `potential_leads.db` кнопка предлагает выбрать реальную ситуацию из списка (уже было реализовано, проверено). |
| **Время публикаций** | В черновиках отображается «🕐 Время публикации: DD.MM HH:MM». Новые посты по умолчанию создаются с временем 12:00. |

## Деплой

- **Репозиторий:** изменения закоммичены и запушены в `main` (коммит 5b64025).
- **Сервер:** выполнен `git pull`, `pm2 restart anton-2-bot`.
- **Логи:** после рестарта нет ImportError, SyntaxError. Бот стартует, каналы TG и рабочая группа OK. В логах есть предупреждение по VK (группа не найдена — проверить VK_GROUP_ID в .env) и ошибки подключения к Router AI (сеть/DNS на сервере). Конфликты getUpdates в логах относились к предыдущему экземпляру; после рестарта работает один процесс.

## Подтверждение для Юлии

**Изменения внесены в репо, деплой на сервер выполнен.** В логах после рестарта критичных ошибок (ImportError, SyntaxError, 401/404) нет. Рекомендуется проверить вручную: переход по ссылке квиза, генерацию по фото, ИИ-Визуал, кнопки VK и «Во все каналы», выбор темы для «Интересный факт» из списка.
