# Отчет о проделанной работе: Ребрендинг и Оптимизация "ЛАД В КВАРТИРЕ"

## 1. Ребрендинг (LAD V KVARTIRE)
*   **Название:** Все упоминания "Пархоменко и компания" и "AI Consultant" заменены на **"ЛАД В КВАРТИРЕ"**.
*   **Персонаж:** Бот Антон теперь является **Ассистентом (Lead Assistant)**, а не Консультантом.
*   **Ссылки:** Ссылки на соцсети обновлены (VK: `vk.com/lad_v_kvartire`).
*   **Landing Page:** Файл `updated_landing.html` обновлен с новым дизайном и логотипом.
*   **Прайс-лист:** Создан актуальный `knowledge_base/10_Price_List.md`.

## 2. Изменение логики бота (`bot_unified.py`)
*   **Упрощенная регистрация:** Удален этап подтверждения имени. Сразу после получения контакта данные сохраняются в `contacts.json`, стейт сбрасывается, и вызывается главное меню.
*   **Отключение RAG для пользователей:** Прямые вопросы к базе знаний через ИИ временно отключены для обычных пользователей. Все запросы на консультацию перенаправляются на Квиз (сбор лидов).
*   **Админ-панель (Изображения):** Добавлена возможность генерации изображений через Yandex Art прямо из Telegram (для админов в группе лидов).

## 3. Автоматизация контента
*   **Yandex Art Интеграция:** Создан `agents/image_agent.py` для генерации картинок по текстовым промптам.
*   **Storytelling:** `content_agent.py` настроен на генерацию постов в стиле сторителлинга с автоматическим созданием промптов для картинок.
*   **Auto-posting:** `auto_poster.py` теперь поддерживает отправку локально сгенерированных фото (`send_photo`).

## 4. База данных
*   **Миграции:** `database.py` автоматически добавляет колонки `image_prompt` и `image_url` при запуске.

## 5. Инструкция по деплою на Timeweb
1. Сделайте `git pull` (или используйте созданный `deploy.sh`).
2. Убедитесь, что в `.env` (или `config`) прописаны ключи:
   - `TELEGRAM_TOKEN`
   - `YANDEX_API_KEY`
   - `YANDEX_CATALOG_ID`
3. Запустите `./deploy.sh`.
